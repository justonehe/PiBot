# 🍓 树莓派 4B 具身智能 (Tangible AI) 项目推荐

树莓派 4B (RPi 4B) 是绝佳的具身智能中枢。它有足够的算力运行轻量级 AI 模型（如 Whisper, TinyLlama），同时拥有丰富的 GPIO 接口连接物理世界。

针对你的背景（**教育者 + 生物黑客 + 效率控**），我为你策划了三个不同维度的项目：

---

## 🧘 项目一："Deep Work" 实体控制台 (The Flow Console)

> **核心概念**：将抽象的"进入心流"动作，具象化为物理开关。

- **硬件组成**：
  - 树莓派 4B
  - **大号物理拨杆开关** (3D打印复古风格) 或 **机械键盘轴体**
  - **e-Paper 墨水屏** (显示当前任务/倒计时，不发光不分心)
  - RGB 灯带 (状态指示)
- **功能逻辑**：
  1.  **物理启动**：当你拉下拨杆，RPi 调用 API 触发 "Focus Mode"。
  2.  **环境联动**：自动通过 Home Assistant 关闭房间大灯，打开台灯，播放白噪音。
  3.  **AI 拦截**：RPi 运行一个简单的本地 LLM (如 Gemma 2b)，拦截手机/电脑通知，只在墨水屏上显示"极高优先级"的信息摘要。
- **技术栈**：Python (GPIO Zero), Home Assistant API, Ollama (本地跑小模型)。

---

## 🏫 项目二：课堂"魔法"助教 (Classroom Totem)

> **核心概念**：一个放在讲台上的物理图腾，通过实体交互辅助教学管理。

- **硬件组成**：
  - 树莓派 4B + **摄像头模块**
  - **NFC 读卡器**
  - **旋转编码器** (旋钮)
  - 小型扬声器
- **功能逻辑**：
  1.  **作业提交**：学生将作业本放在摄像头下，按下按钮。RPi 拍照 ->OCR -> LLM 快速分析 -> 朗读一句简短的鼓励评语 ("代码结构很清晰，Joe！")。
  2.  **状态切换**：通过旋转旋钮切换课堂模式（讲课/讨论/自习）。不同模式下，图腾显示不同颜色的光环，并自动调整教室屏幕显示的倒计时或背景音乐。
  3.  **实物投票**：学生用 NFC 卡片触碰图腾进行投票或抢答。
- **在信奥课的应用**：作为算法可视化的物理载体。比如并在排序算法运行时，图腾发出不同频率的声音。
- **技术栈**：OpenCV, Whisper (语音识别), Piper (本地TTS)。

## 🧠 项目三：生物反馈环境舱 (Bio-Feedback Pod)

> **核心概念**：感知环境并将你的生理数据"外化"的桌面设备。

- **硬件组成**：
  - 树莓派 4B
  - **CO2 传感器 (SCD40)** (二氧化碳浓度直接影响认知能力)
  - **温湿度传感器**
  - **伺服电机 (Servo)** (用于物理动作)
- **功能逻辑**：
  1.  **认知保护**：当 CO2 > 1000ppm 时，伺服电机驱动一个小旗子升起，或者敲击一下桌子，物理提醒你"该开窗换气了，你的智商正在下降"。
  2.  **HRV 可视化**：对接你的 Garmin/Apple Health 数据。当你压力过大 (HRV 低) 时，设备不仅显示红灯，还可以**拒绝**执行非紧急的脚本任务，强迫你休息。
  3.  **睡眠建议**：根据温湿度历史数据，结合你昨晚的睡眠分数，每天早晨给出物理提示（如打印一张小纸条）。
- **技术栈**：InfluxDB (数据存储), Garmin API 抓取。

---

## 🚀 推荐起步：项目一 (Flow Console)

**理由**：

1.  **立竿见影**：直接服务于你的"生产力"和"写作"。
2.  **硬件简单**：只需要几个开关和屏幕，不需要复杂的机械结构。
3.  **代码复用**：你会用到 Python 脚本控制，这正是你擅长的。
